# Sparkwyrd

A Tcl/Tk reimplementation of the **Inspiration Pad Pro** random text generator, compatible with `.ipt` generator files.

Sparkwyrd reads `.ipt` files — plain-text rule files containing weighted tables, dice expressions, variables, and filters — and generates randomised narrative content. Typical uses: RPG encounter generators, NPC creators, treasure tables, name generators, story seeds.

---

## Quick start

**Single-file build (recommended):**
```sh
make                              # build sparkwyrd.tcl from src/
./sparkwyrd.tcl --gui             # launch GUI
./sparkwyrd.tcl --gui demo.ipt    # launch GUI and open a file
./sparkwyrd.tcl --cli demo.ipt    # one generation, plain text
./sparkwyrd.tcl --cli -n 5 names.ipt
```

**Run source files directly:**
```sh
wish  src/gui.tcl                 # GUI
wish  src/gui.tcl my_gen.ipt      # GUI with file
tclsh src/cli.tcl --cli demo.ipt  # CLI
```

The GUI auto-loads `demo.ipt` on startup if it is in the same directory as the script.

## CLI options

```
./sparkwyrd.tcl [--gui] [file.ipt]        launch GUI (default)
./sparkwyrd.tcl --cli [options] file.ipt  command-line mode

Mode:
  --gui           Open graphical interface (default when no CLI flags present)
  --cli           Force command-line mode

Options (CLI mode):
  -n <count>      Number of generations (default: 1)
  -t <table>      Start from a named table instead of the first one
  --html          Keep HTML tags in output (default: tags are stripped)
  --sep <string>  Separator between outputs (default: blank line)
  --list          List all table names in the file and exit
  --version       Show version and exit
  --help          Show this help and exit
```

HTML tags are **stripped by default** in CLI mode. Use `--html` to preserve them.

## .ipt file format — quick reference

Generator files are plain UTF-8 text. The **first table** is the entry point.

```
# Comments: #  ;  // (line-start or inline)

Header: <b>My Generator</b>     displayed above output (HTML allowed)
Footer: Generated by Sparkwyrd  displayed below output
MaxReps: 5                       cap on repetitions
Use: lib/names.ipt               import tables from another file

Set: world = [@WorldNames]       global variable (evaluated once at start)

Table: MyTable
Set: hero = [@HeroNames]         directive: always runs, not a random item
3:A common result                weight 3
A normal result                  weight 1 (default)
\z                               explicit empty result
long item split &
across two source lines          & at line-end = continuation
```

### Syntax elements

| Element | Meaning |
|---|---|
| `[@Table]` | Random roll on Table |
| `[@N Table]` | N random rolls |
| `[#N Table]` | Pick item at index N (1-based) |
| `[!N Table]` | N deck picks (no repeats) |
| `[|a|b|c]` | Random inline choice (equal probability) |
| `{1d6}` `{2d10+3}` | Dice roll |
| `{1d6 * 10}` | Dice + math expression |
| `{varname}` `{$varname}` | Variable reference |
| `{x=expr}` / `{x==expr}` | Assign + output / assign silently |
| `[@x=Table]` / `[@x==Table]` | Roll + assign + output / silently |
| `[@Table >> filter]` | Apply filter to result |
| `[@Table with p1, p2]` | Pass parameters; use `{$1}` `{$2}` inside Table |
| `[when]c[do]e[end]` | Conditional |
| `[when]c[do]e[else]f[end]` | Conditional with else |
| `[when not]c[do]e[end]` | Negative conditional |
| `\n` `\t` `\z` `\_` `\a` | Newline / tab / empty / forced space / a or an |
| `&` at line end | Continue item on next line |

### Filters (after `>>`, chainable)

| Filter | Effect |
|---|---|
| `lower` `upper` `proper` | Case conversion |
| `bold` `italic` `underline` | HTML formatting (`<b>` etc.) |
| `sort` | Sort multiple results alphabetically |
| `implode [sep]` | Join multiple results (default separator: `, `) |
| `trim` `ltrim` `rtrim` | Strip whitespace |
| `length` `reverse` | String length / reversal |
| `left N` `right N` `substr S L` | Substrings |
| `at sub` | Position of substring (0 if absent) |
| `replace /find/repl/` | Text replacement |
| `+-` | Format number with explicit sign (+3, -1, +0) |
| `each Table` | Pass each result to Table as `{$1}` |
| `eachchar Table` | Pass each character to Table as `{$1}` |

### Use: — importing external tables

```
Use: lib/names.ipt
Use: /absolute/path/to/tables.ipt
```

- Tables are searched relative to the current file's directory, then the working directory.
- The main file's tables always take priority over imported ones.
- Circular imports are detected and skipped automatically.
- A warning is printed (not an error) if the file is not found.

## Build

```sh
make        # produces sparkwyrd.tcl (engine + cli + gui merged)
make clean  # removes sparkwyrd.tcl
```

The build concatenates `src/engine.tcl`, `src/cli.tcl`, `src/gui.tcl`, and `src/main.tcl` into a single executable file with a sh/Tcl polyglot header. The result runs with `tclsh` (CLI default) or `wish` (GUI).

---

## Requirements

- Tcl 8.5 or later
- Tk 8.5 or later (GUI only — CLI and engine work with `tclsh` alone)

---

## Compatibility

Sparkwyrd is compatible with the `.ipt` format used by **Inspiration Pad Pro** by NBOS Software. See `manual.md` for the full syntax reference and feature status.
