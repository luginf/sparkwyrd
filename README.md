# Sparkwyrd

A Tcl/Tk reimplementation of the **Inspiration Pad Pro** random text generator, compatible with `.ipt` generator files.

Sparkwyrd reads `.ipt` files — plain-text rule files containing weighted tables, dice expressions, variables, and filters — and generates randomised narrative content. Typical uses include RPG encounter generators, NPC creators, treasure tables, name generators, and story seeds.

---

## Quick start

**GUI (requires Tk / wish):**
```
wish sparkwyrd.tcl
wish sparkwyrd.tcl my_generator.ipt
```

**CLI (requires tclsh only):**
```
tclsh cli.tcl my_generator.ipt
tclsh cli.tcl -n 5 names.ipt
tclsh cli.tcl -n 3 --plain adventure.ipt
tclsh cli.tcl --list demo.ipt
```

The GUI auto-loads `demo.ipt` on startup if it is in the same directory.

## CLI options

```
tclsh cli.tcl [options] file.ipt

  -n <count>      Number of generations (default: 1)
  -t <table>      Start from a named table instead of the first one
  --plain         Strip HTML tags from output
  --sep <string>  Separator between outputs (default: blank line)
  --list          List all table names in the file and exit
  --version       Show version
  --help          Show help
```

## .ipt file format — quick reference

Generator files are plain UTF-8 text. The **first table** in the file is the entry point.

```
# Comments: #  ;  // (line-start or inline)

Header: <b>My Generator</b>     ← displayed above output (HTML allowed)
Footer: Generated by Sparkwyrd  ← displayed below output
MaxReps: 5                       ← maximum repetitions allowed

Set: varname = [@SomeTable]      ← global variable (evaluated once)

Table: MyTable
Set: hero = [@HeroNames]         ← directive: always runs before pick
3:A common result                ← weight 3
A normal result                  ← weight 1 (default)
\z                               ← explicit empty result

Table: HeroNames
Aldric
Sylvara
Torinn
```

### Syntax elements

| Element | Meaning |
|---|---|
| `[@Table]` | Random roll on Table |
| `[@N Table]` | N random rolls |
| `[#N Table]` | Pick item at index N (1-based) |
| `[!N Table]` | N deck picks (no repeats) |
| `[|a|b|c]` | Random inline choice |
| `{1d6}` `{2d10+3}` | Dice roll |
| `{1d6 * 10}` | Dice + math |
| `{varname}` `{$varname}` | Variable reference |
| `{x=1d6}` | Assign + output |
| `{x==1d6}` | Assign silently |
| `[@x=Table]` | Roll + assign + output |
| `[@x==Table]` | Roll + assign silently |
| `[@Table >> filter]` | Filtered roll |
| `[when]c[do]e[end]` | Conditional |
| `[when]c[do]e[else]f[end]` | Conditional with else |
| `[@T with p1,p2]` | Pass parameters; use `{$1}` `{$2}` in T |
| `\n` `\t` `\z` `\_` `\a` | Newline / tab / empty / space / a–an |
| `&` at line end | Continue item on next line |

### Filters (after `>>`)

`lower` `upper` `proper` `bold` `italic` `underline` `sort` `implode` `trim` `length` `left N` `right N` `substr S L` `at sub` `replace /f/r/` `reverse` `+-` `each T` `eachchar T`

## Requirements

- Tcl 8.5 or later
- Tk 8.5 or later (GUI only; CLI requires tclsh only)

---

## Compatibility

Sparkwyrd is compatible with the `.ipt` format used by **Inspiration Pad Pro** by NBOS Software. Most features of the format are supported; see `manual.md` for the complete list and any known limitations.
